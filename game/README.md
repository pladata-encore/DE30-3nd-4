# PYGAME 구현
파이게임 사이트 : https://www.pygame.org/news
 - Pygame 및 필요한 라이브러리 설치
    - pip install pygame
    - pip install PIL
    - pip install requests

    - import random
    - import math
    - import requests
    - from PIL import Image
    - from datetime import datetime
